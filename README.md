# Minecraft Server 1.20.1 (Docker)
### Instalation
- Clone this project
```bash
git clone https://github.com/surattinon/mc-docker-1.20.1
cd mc-docker-1.20.1
```
### Command to operate Server
- Start
```bash
./start.sh
```
- Stop
```bash
./stop.sh
```
- Restart
```bash
./restart.sh
```
- Attach Docker container
```bash
./attach.sh
```
